# Command History

Raycast Extension zum schnellen Durchsuchen der `~/.zsh_history` mit `fzf`.

## Features

- Liest deine `~/.zsh_history` ein (inkl. zsh extended history format).
- Durchsuchung per `fzf` bei jeder Sucheingabe.
- Action zum direkten Einfuegen in die aktive Anwendung.
- Zusaetzliche Action zum Kopieren in die Zwischenablage.

## Voraussetzungen

- `fzf` muss installiert und im `PATH` verfuegbar sein.
- Installation auf macOS z. B. mit:

```bash
brew install fzf
```
